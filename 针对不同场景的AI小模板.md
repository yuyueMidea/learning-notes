好的prompt一定包含以下的
```
角色
背景
任务
约束
输出结构
自检
```

1、让AI写功能
```
我需要实现以下功能：

技术栈：
文件结构：

请提供：

1 实现思路
2 完整代码
3 如何集成到现有项目
```

2、让AI debug
```
下面代码运行报错：

报错信息：
代码：

请：

1 找出 bug
2 解释原因
3 给出修复后的代码
```

3、让AI重构代码
```
请重构以下代码：

要求：

- 提高可读性
- 减少重复
- 遵循最佳实践
```

4、让AI写复杂组件
```
请实现一个组件：

功能：
技术栈：

要求：

- 完整代码
- 示例用法
```

举例说明，之前做的 React 表单设计器项目，高质量的prompt:
```
我正在开发一个 React 表单设计器。

技术栈：

- React
- TypeScript
- 不使用 UI 框架
- 使用原生 drag 实现拖拽

当前功能：

- 左侧组件库
- 中间画布
- 右侧属性编辑

我希望新增：

1 JSON Schema 导入导出
2 表单预览模式
3 表单数据提交

请给出：

1 整体设计方案
2 关键数据结构
3 React 组件实现
4 示例 JSON Schema
```
